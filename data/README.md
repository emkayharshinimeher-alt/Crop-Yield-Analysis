# Data Description

## Source
Iowa crop yield dataset provided for MHYWI05 course (TU Dresden)

## Overview
- **Time period:** 115 years
- **Locations:** 25 sites across Iowa
- **Observations:** 2,875 data points
- **Target variable:** Crop yield

## Predictors (23 meteorological variables)
- **Precipitation:** pr_1 through pr_6 (6 variables)
- **Radiation:** rad_1 through rad_3 (3 variables)  
- **Temperature:** temp_1 through temp_4 (4 variables)
- **Extreme indices:** 
  - warm_days
  - max_temp
  - max_5day_pr
  - [Additional extreme event indicators]

## Data Privacy
Original dataset not publicly shared per course policy. 
Data available through MHYWI05 course at TU Dresden.

## Statistics
- **Mean yield:** 9.55
- **Standard deviation:** 2.84
- **Distribution:** Approximately normal with slight left skew
